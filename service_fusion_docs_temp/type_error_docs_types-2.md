# Type: Error

**Description:**
An error's schema.

**Schema Definition:**
```json
{
  "code": "integer",
  "name": "string",
  "message": "string"
}
```

**Parameters:**

| Parameter | Type    | Description                               |
| :-------- | :------ | :---------------------------------------- |
| `code`    | Integer | The error code associated with the error. |
| `name`    | String  | The error name associated with the error. |
| `message` | String  | The error message associated with the error. |
